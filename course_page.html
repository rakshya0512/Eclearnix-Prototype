Python Data Types

Python has several built-in data types. Here‚Äôs a summary with examples:

1. Numeric Types

Integers and floats for numbers.

x = 10       # integer
y = 3.14     # float
print(x + y) # 13.14

2. String

Sequence of characters.

name = "Alice"
print("Hello " + name)  # Hello Alice

3. Boolean

True or False values.

is_active = True
print(not is_active) # False

4. List

Ordered collection.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple

5. Tuple

Immutable ordered collection.

colors = ("red", "green", "blue")
print(colors[1])  # green

6. Dictionary

Key-value pairs.

person = {"name": "Bob", "age": 25}
print(person["name"])  # Bob

7. Set

Unordered collection of unique elements.

numbers = {1, 2, 3, 2}
print(numbers)  # {1,2,3}
